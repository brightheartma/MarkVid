【核心工作流】
═══════════════════════════════════════════════════════════════════════════

  1️⃣  准备视频
      ↓
  2️⃣  添加到 data/input/     ← bash scripts/add_video.sh
      ↓
  3️⃣  执行流水线              ← bash scripts/run_pipeline.sh
      ↓
      ├─ 步骤 1：提取媒体      python src/extract_media.py
      │           ├─ 输入：data/input/*.mp4
      │           └─ 输出：data/output/{video_name}/audio.mp3
      │                      data/output/{video_name}/frame_*.jpg
      │
      ├─ 步骤 2：转录          python src/transcribe_audio.py
      │           ├─ 使用 Groq API
      │           ├─ 自动分片（文件>18MB）
      │           └─ 输出：data/output/{video_name}/transcript/{audio.srt, transcript.md}
      │
      ├─ 步骤 3：生成笔记      python src/generate_note.py --all
      │           ├─ 使用 Groq LLM（llama-3.3-70b）
      │           ├─ 自动合并同课程的多个视频段
      │           └─ 输出：notes/{course_name}.md + INDEX.md
      │
      └─ 步骤 4：清理          rm -rf data/output/*/transcript/_chunks_work/
      ↓
  4️⃣  查看笔记
      └─ notes/{course_name}.md
      └─ notes/INDEX.md

【快速命令速查】
═══════════════════════════════════════════════════════════════════════════

┌─ 📥 添加视频 ────────────────────────────────────────────────────────────┐
│                                                                            │
│  # 方式 A：自动复制 + 触发处理（推荐）                                   │

└────────────────────────────────────────────────────────────────────────────┘

┌─ 🚀 执行流水线 ──────────────────────────────────────────────────────────┐
│                                                                            │

│  bash scripts/run_pipeline.sh                                             │
│                                                                            │
│  # 强制重新生成所有笔记                                                  │
│  bash scripts/run_pipeline.sh --force-note                               │

│  # 跳过某些步骤                                                          │
│  bash scripts/run_pipeline.sh --skip-extract       # 仅转录+笔记         │
│  bash scripts/run_pipeline.sh --skip-transcribe    # 仅提取+笔记         │
│  bash scripts/run_pipeline.sh --note-only          # 仅笔记              │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘

┌─ 🔧 单步执行 ─────────────────────────────────────────────────────────────┐

【关键特性】
═══════════════════════════════════════════════════════════════════════════

  ✅ 增量处理
     - 已提取的视频自动跳过（extract_media.py）
     - 已转录的音频自动跳过（transcribe_audio.py）
     - 已生成的笔记自动跳过（generate_note.py）
     - 使用 --force 或 FORCE=true 可覆盖

  ✅ 自动分片（超大文件处理）
     - 文件 >18MB 时自动分片（5分钟/片）
     - 自动处理 413 错误和超时
     - 分片后重新合并完整转录

  ✅ 课程自动合并
     - 多个同课程的视频段（00_14_第十四课_01/02/03...）
     - 自动合并生成一份课程笔记（00_14_第十四课：xxx.md）
     - 按视频顺序排列内容

  ✅ INDEX.md 自动维护
     - 每次生成笔记后自动更新
     - Wiki 链接格式 [[笔记名]]
     - 包含课程、段数、时长、转录行数、关键帧数统计
